# score-post-notifier

This is a Discord bot for notifying channels about new score posts on
[/r/osugame](https://reddit.com/r/osugame), especially useful for video
uploaders.

### Features

* Post in any channel
* Tag any role/user
* Submitter blacklist
* Player blacklist
